jQuery.googleslides

A jQuery plugin to display your Google Photos, including Picasa and Google+ albums.

Demos

Demo 1

		$("#demo1").picasaSlides( {
		  user : '115528839112598673902', 
		  albumid: '5708733709522191137'
		});

Demo 2

		$("#demo2").picasaSlides( {
		  user : '115528839112598673902', 
		  albumid: '5708746270943603681', 				
		  imgmax: 100,
		  fadespeed: 500,
		  time: 8000,
		  caption : false
		});

Demo 3

		$("#demo3").picasaSlides( {
		  user : '115528839112598673902', 
		  albumid: '5708757017692613217', 				
		  time:2000,
		  imgmax: 260,
		  loadingimage: false
		});
