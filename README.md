# jQuery.googleslides  
A jQuery plugin to display your Google Photos, including Picasa and Google+ albums.

## Basic Usage
        $("#slidesContainer").googleslides({
           userid:'115528839112598673902', 
           albumid:'5710317752556741025'
        });
